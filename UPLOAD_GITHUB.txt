# Cara Upload ke GitHub untuk Loadstring

## Langkah 1: Install Git (jika belum)
Download dari: https://git-scm.com/download/win

## Langkah 2: Setup Git (jalankan di PowerShell/CMD)
git config --global user.name "NamaKamu"
git config --global user.email "emailkamu@example.com"

## Langkah 3: Buat Repository di GitHub
1. Buka https://github.com
2. Klik tombol "+" > "New repository"
3. Nama repository: bypatih (atau terserah)
4. ⚠️ PENTING: Set ke PUBLIC (jangan private!)
5. Jangan centang "Add a README file"
6. Klik "Create repository"

## Langkah 4: Upload File (jalankan di folder PROJECTCODING\bypatih)
cd C:\PROJECTCODING\bypatih
git init
git add vbypatih.lua
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/USERNAME/REPONAME.git
git push -u origin main

## Langkah 5: Dapatkan Raw URL
Setelah upload berhasil:
1. Buka file di GitHub (vbypatih.lua)
2. Klik tombol "Raw" di kanan atas
3. Copy URL nya, formatnya:
   https://raw.githubusercontent.com/USERNAME/REPONAME/main/vbypatih.lua

## Langkah 6: Pakai di Delta Executor
loadstring(game:HttpGet("https://raw.githubusercontent.com/USERNAME/REPONAME/main/vbypatih.lua"))()

## Tips:
- Ganti USERNAME dengan username GitHub kamu
- Ganti REPONAME dengan nama repository kamu
- File HARUS di branch "main"
- Repository HARUS PUBLIC
